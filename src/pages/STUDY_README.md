# 🎯 React サイドバー学習プロジェクト

このプロジェクトは、Reactを使って高度なサイドバー機能を段階的に学習するための教材です。

## 🚀 学習の進め方

### 1. 学習ページにアクセス
```bash
# 開発サーバーを起動
npm run dev

# ブラウザで以下にアクセス
http://localhost:5173/study
```

### 2. 段階的な学習ステップ

#### STEP 1: 基本メニューアイテム 📋
- Material-UIアイコンの使用
- メニューアイテムの配列作成
- 基本的なクリックハンドリング
- **所要時間**: 15分

#### STEP 2: ホバー状態管理 🖱️
- useStateフックの使用
- マウスイベントハンドリング
- React DevToolsでの状態確認
- **所要時間**: 20分

#### STEP 3: 幅アニメーション 📐
- CSS transitionプロパティ
- 動的スタイル適用
- スムーズなアニメーション実装
- **所要時間**: 25分

#### STEP 4: テキストアニメーション ✨
- 複数プロパティのアニメーション
- opacityとtransformの組み合わせ
- タイミングの調整
- **所要時間**: 30分

#### STEP 5: レイアウト安定化 🎯
- 固定幅レイアウトの実装
- アニメーション中のずれ防止
- overflowプロパティの活用
- **所要時間**: 25分

#### STEP 6: 設定ページ機能 ⚙️
- 複数状態の管理
- 条件分岐レンダリング
- 3段階幅変更の実装
- **所要時間**: 35分

**総学習時間**: 約2.5時間

## 🛠️ 学習方法

### A. コメントアウトを解除する方法
1. `src/pages/StudyPage.tsx`を開く
2. 各STEPのコメントアウト(`// `)を段階的に外す
3. ブラウザで動作確認
4. チェックリストで完了確認

### B. ゼロから実装する方法
1. `src/components/StudySidebar.tsx`を新規作成
2. `StudyGuide.ts`の手順に従って実装
3. 各ステップ完了時にチェックリスト確認

## 🚨 トラブルシューティング

### よくあるエラー

#### 1. モジュールが見つからない
```bash
# Material-UIアイコンがない場合
npm install @mui/icons-material

# React関連のエラー
npm install react@latest react-dom@latest
```

#### 2. TypeScriptエラー
```typescript
// 型定義が不明な場合
const menuItems: { icon: React.ReactNode; text: string; onClick: () => void }[] = [
  // ...
];
```

#### 3. スタイルが適用されない
```typescript
// sx propを使用
<Box sx={{ width: sidebarWidth, transition: 'width 0.3s ease' }}>
```

### デバッグ方法
1. **React DevTools**で状態確認
2. **ブラウザ開発者ツール**でCSS確認  
3. **console.log**で値確認
4. **チェックリスト**で漏れ確認
